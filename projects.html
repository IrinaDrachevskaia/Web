My Projects

On this page, you'll find several of my projects, along with their descriptions and links. If I were asked which project is my favorite or which one I am most proud of, I wouldn't be able to answer. That's because I love each of my projects; I've invested a part of myself into every one of them, and I cherish each project in its own way. The project I use most frequently is the Password Generator. You can discover more of my projects by following the link provided in my GitHub profile.

Places To Visit In England

This is a blog where you can find recommendations for places to visit in England with descriptions. No database was utilized; all data is refreshed upon reloading the site. Users have the ability to add or delete posts, make changes to descriptions, and leave comments. All pages are responsive, ensuring a seamless experience on both desktop and mobile devices.

Programming language:

JavaScript, HTML, CSS.

Used libraries and modules:

NodeJs, ExpressJs
